- **버전**: v1.0 (MVP)

## 1. 개요 (Overview)

### 1.1 배경 및 문제 정의

건설사업관리(CM) 용역사는 발주처에 기성을 청구할 때 현장 직접경비(숙식비, 교통비, 현장운영비 등)를 정산하여 증빙과 함께 제출하여야 한다. 현재 선엔지니어링은 다음과 같은 아날로그 방식에 의존하고 있다.

- 현장 직원이 엑셀로 비용 내역을 수기 작성
- 종이 영수증 또는 이미지 파일을 별도 수집하여 본사에 우편·이메일 제출
- 본사 담당자가 수작업으로 규정 적합성 검토(공무원 여비규정, 건설엔지니어링 대가 기준 등)
- 반려 시 현장 재제출 → 기성 청구 지연 반복 발생

이로 인해 발생하는 핵심 문제:

- **증빙 누락**: 항목별 필수 증빙자료(세금계산서, 출근부 등) 미제출로 발주처 반려
- **규정 위반**: 식대 2.5만원 한도, 복리후생비 5만원/인·월 한도 등 공무원 여비규정 초과 입력
- **정산 지연**: 기성 청구 마감 전 수작업 검토에 2~3일 소요
- **데이터 분산**: 프로젝트별 비용 데이터가 개인 PC에 산재하여 이력 추적 불가

### 1.2 제품 목적

| 목표      | 내용                 | KPI                      |
| ------- | ------------------ | ------------------------ |
| 업무 효율화  | 정산서 작성 및 검토 시간 단축  | 정산서 작성 시간 80% 단축 (기존 대비) |
| 오류 제거   | 시스템이 규정 준수를 자동 검증  | 규정 위반으로 인한 반려율 0건 목표     |
| 데이터 자산화 | 프로젝트별 비용 데이터 누적 관리 | 전사 11개 이상 현장 데이터 통합 조회   |

### 1.3 제품 범위

- **대상 조직**: 선엔지니어링 건설사업관리본부 전 현장 (11개 이상)
- **1단계(MVP)**: 핵심 직접경비 항목 입력·검증·정산서 출력
- **2단계**: 기성 청구 패키지 생성, 발주처 제출용 공식 출력물 완성
- **제외 범위**: 발주처 시스템 직접 연동, 급여·노무비 정산, ERP 연동, 회계 시스템 연계 (추후 검토)

### 3.2 기타사항 (시스템 적용 원칙)

- 항목별 비용은 직접경비 내에서 항목 간 이동(변경)이 가능하며, 발주청은 공사 및 지역 특성을 고려하여 직접경비를 증·감할 수 있다.
- 발주청은 정산 내용에 맞게 사용된 금액에 대해 별도 증빙자료를 검토하여 인정·정산할 수 있다.
- 발주청은 직접경비의 사용금액이 항목별 목적에 부합하지 아니할 경우 해당 금액을 불인정하여야 한다.
- 시스템은 한도 초과 입력 시 즉시 경고를 표시하고, 초과분은 자동으로 표시 분리하여 정산 대상에서 제외한다.

### 5.1 UI/UX

- 현장 직원의 IT 숙련도가 낮음을 전제로 3-클릭 이내 주요 기능 접근 구현
- 모바일 반응형 웹 지원 (현장에서 스마트폰으로 영수증 즉시 업로드 가능)
- 비목 선택 후 필수 증빙 안내가 폼 내에 즉각 표시되어 '몰라서 누락' 방지
- 오류 메시지: 무엇이 왜 잘못됐는지, 어떻게 수정하면 되는지 구체적으로 안내

### 5.2 성능

- 페이지 로딩: 3초 이내 (일반 LTE 환경 기준)
- 영수증 업로드: 10MB 이하 파일 업로드 5초 이내 완료
- 정산서 출력(엑셀·PDF): 30초 이내 생성
- 동시 사용자 50명 이상 처리 가능

### 5.3 보안

- HTTPS 통신 필수 (TLS 1.2 이상)
- 역할 기반 접근 제어(RBAC): 타 현장 데이터 접근 차단
- 영수증 파일 저장소 접근은 서버 서명 URL(Pre-signed URL) 방식 적용
- 금융 정보·계약 관련 민감 데이터 암호화 저장 (AES-256 이상)
- 세션 타임아웃: 비활성 30분 후 자동 로그아웃

### 5.4 규정 유연성

- 현장별 정산 기준 파라미터(식대 한도, 복리후생비 한도, 여비규정 등급 등)를 관리자가 UI에서 직접 변경 가능
- 법령 개정 시 파라미터 업데이트만으로 시스템 전체 적용 가능한 구조

### 5.5 데이터 보존

- 정산 데이터 및 영수증: 최소 5년 보존 (「국가계약법」 계약 서류 보존 기간 기준)
- 삭제는 논리 삭제 방식으로, 데이터 복구 가능

## 9. 용어 정의 (Glossary)

| 용어          | 정의                                                                |
| ----------- | ----------------------------------------------------------------- |
| 직접경비        | 건설사업관리 용역 수행에 직접 소요되는 비용 (현장주재비, 차량운행비, 출장비 등). 간접비·이윤과 구분됨       |
| 기성          | 계약된 용역의 수행 실적에 따라 발주처에 청구하는 대가. 월별 또는 공정 단계별로 청구                  |
| 실비 적용       | 실제 지출한 금액을 영수증 증빙 기반으로 정산하는 방식 (한도 없음, 단 목적 부합 필수)                |
| 공무원 여비규정    | 출장비, 교통비, 일비 등의 한도 및 산정 기준을 규정한 법령. 건설엔지니어링 직접경비의 교통비·출장비 기준으로 준용 |
| 불인정         | 정산 기준에 부합하지 않아 발주청이 해당 금액을 정산 대상에서 제외하는 것                         |
| CM (건설사업관리) | Construction Management. 발주청을 대리하여 공사 전반의 기획·설계·시공·유지관리를 관리하는 용역  |
| MVP         | Minimum Viable Product. 핵심 기능만으로 구성된 최초 출시 버전                     |
| RBAC        | Role-Based Access Control. 사용자 역할에 따라 시스템 접근 권한을 차등 부여하는 보안 방식    |
